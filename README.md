# Lightweight PHP Root Manager for Educational Research

Welcome to the YamiRoot Project repository. This is a collection of lightweight PHP scripts (File Manager, Helper Tools) designed for learning and research in cybersecurity.

**visit:** **[yamiroot.github.io](https://yamiroot.github.io/)**

---


## üìö About This Project

This project demonstrates the functionality and mechanisms behind server management tools and webshells within a controlled PHP environment. **It features a collection of different series (e.g., Mini, Bypass, RichTools), each designed for specific purposes,** along with various helper modules.

It originally evolved from the [0x6NyxWebShell](https://github.com/6ickzone/0x6NyxWebShell) project by the same author.

## ‚ú® Key Features

* **YamiRoot Series:**  YamiRoot Series Collection.
* **Helper Tools:** Mock-ups for integrated tools, including:
    * Mass Deface
    * Jumping
    * Reverse Shell
    * Network Scanners (Ping, Port Scan)
* **Collections:** Educational articles explaining the inner workings of popular webshells (like WSO, R57, etc.).
